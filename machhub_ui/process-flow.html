Manufacturing order workflow

제조 가공 발주 플랫폼 흐름도

클라이언트, 관리자, 가공업체가 같은 발주를 어떤 단계에서 주고받는지 한 화면에서 확인합니다.

클라이언트 화면 관리자 화면 업체 화면
클라이언트
관리자 / 플랫폼
가공업체
1
회원가입 / 기본정보 구축
  • 이름, 연락처, 회사정보, 이메일 등록
  • PASS 인증으로 계정 신뢰도 확보
  • 클라이언트·업체 DB 관리
  • 발주 단계, 거래 내역, 평가, 문의 상태 추적
  • 회사명, 주소, 사업자등록번호 등록
  • 규모, 재질, 서비스, 장비, 스케줄 입력
2
견적 요청 등록
  • 도면 파일, 납기, 수량, 수령방식, 위치 입력
  • 용도, 예산, 양산 계획, 후처리, 품질 인증 요청
  • 견적 요청서 생성
  • 파일·도면·입력값 수집 및 접수 상태 관리
  • 오더 수신 전 대기
  • 실시간 작업 가능 상태 유지
3
AI 분석 및 정보 보완
  • AI가 요청한 추가 정보 입력
  • 누락 치수, 수령 방식, 품질 조건 보완
  • 도면 이미지·텍스트 인식
  • 치수, 수량, 공차 등 중요 누락 정보 탐지
  • 보완 완료 후 매칭 대상 여부 결정
4
업체 매칭 및 오더리스트 발송
  • 보완 완료 후 매칭 대기
  • 업체 추가 질의가 있으면 응답
  • 가공종류, 크기, 스케줄 기준 업체 선별
  • 클라이언트 평점과 리뷰를 업체에 제공
  • 오더리스트 검토 및 수락 여부 결정
  • 추가 정보 필요 시 관리자에 요청
5
작업정보 회신 및 비교
  • 업체별 금액, 납기, 리뷰, 평점 비교
  • 최종 후보 업체 선정
  • 업체 작업정보를 클라이언트에게 전달
  • 업체 평점과 별점 정보 함께 제공
  • 예상 납기와 금액 회신
  • 작업 가능 범위와 품질 조건 명시
6
발주 / 계약 / 결제
  • 최종 업체 선택 및 발주 진행
  • 계약·급행·결제대행 또는 은행 선입금 요청
  • 계약서 양식, SGI, 결제대행 연계
  • NDA·기밀유지 서약 수령 및 전달
  • 발주 확인 수락 및 작업 스케줄 업데이트
  • 계약 가입, 도면·기술 기밀유지 서약 작성
7
생산 / 진행도 관리
  • 가공물 작업 진행도 확인
  • 지연 및 보상 안내 수신
  • 실시간 작업 진행도 시각화 제공
  • 납기 지연 요청 처리 및 보상 정책 적용
  • 원자재 입고, 가공 시작, 후처리, 검사 이벤트 공유
  • 납기 지연 시 관리자에 지연 서비스 요청
8
납품 / 검수 / 정산
  • 제품 수령 후 자체 QC 및 납품 인증 체크
  • 사용 이후 AS와 질문 사항 문의
  • 최종 상태·포장 이미지와 발송 현황 제공
  • 납품 인증 후 결제대행사 또는 은행 정산 요청
  • 최종 상태 이미지와 포장 이미지 발송
  • 택배, 화물차, 특수화물 등 요청 방식으로 출하
9
리뷰 / 평점 / 사후관리
  • 업체 작업에 대한 리뷰와 평점 등록
  • 사후 문의 이력 확인
  • 상호 리뷰·평점 DB 관리
  • 문의, AS, 휴면화 알림 이력 관리
  • 클라이언트에 대한 리뷰와 평점 등록
  • 재발주 가능 고객 관리
